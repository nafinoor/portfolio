## Deployment to GitHub Pages

### Method 1: Quick Deploy (Recommended)

1. Create a new repository on GitHub named `your-username.github.io` (replace `your-username` with your actual GitHub username)
   - For example: `nafi-bin-noor.github.io`

2. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/your-username.github.io.git
   cd your-username.github.io
   ```

3. Copy the `index.html` file to the repository root

4. Commit and push:
   ```bash
   git add index.html
   git commit -m "Initial portfolio website"
   git push origin main
   ```

5. Your site will be live at `https://your-username.github.io`

### Method 2: Deploy to Existing Repository

1. Create a new repository or use an existing one

2. Upload the `index.html` file to the repository

3. Go to repository Settings ‚Üí Pages

4. Under "Source", select the branch (usually `main`) and folder (`/root`)

5. Click Save

6. Your site will be live at `https://your-username.github.io/repository-name`

## Customization

The website is fully self-contained in a single HTML file. You can customize:

- **Colors**: Edit CSS variables in the `:root` section
- **Content**: Update text content directly in the HTML
- **Links**: Update social media and contact links
- **Fonts**: Change Google Fonts imports if desired
